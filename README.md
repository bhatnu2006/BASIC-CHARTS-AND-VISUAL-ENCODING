## 🎯 Aim
To study and implement various basic charts and visual encoding techniques using Python libraries.

## 🛠️ Tools Used
- Python  
- Jupyter Notebook  
- Pandas (for data handling)  
- Matplotlib (for basic plotting)  
- Seaborn (for advanced visualization)  

## 📚 Theory
Data visualization is the graphical representation of information and data.
It helps in understanding complex datasets by presenting them in visual formats such as charts and graphs.

Different types of charts serve different purposes:

- **Line Chart**: Used to show trends over time or continuous data.
- **Bar Chart**: Used to compare values across categories.
- **Histogram**: Represents the frequency distribution of numerical data.
- **Scatter Plot**: Shows the relationship between two variables.
- **Conditional Visualization**: Uses colors or markers to distinguish categories (e.g., pass/fail).

Libraries like Matplotlib and Seaborn provide powerful tools to create these visualizations easily.
Pandas is used to structure and manipulate the dataset before visualization.

## ✅ Advantages
- Makes complex data easy to understand  
- Helps in identifying trends and patterns quickly  
- Improves decision-making  
- Enhances data presentation  
- Supports comparison between variables  

## ❌ Disadvantages
- Misleading visuals if not designed properly  
- Requires proper understanding of tools  
- Over-visualization can confuse users  
- Not suitable for very small datasets  
- May require preprocessing of data
